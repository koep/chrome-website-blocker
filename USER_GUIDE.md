# Pomodoro Timer - Quick Start Guide

## What is This?

Your website blocker now includes a built-in Pomodoro timer! Instead of just showing a motivational quote when you visit a blocked site, you'll see an active timer that helps you maintain focus through structured work sessions.

## How to Use

### 1. Start a Focus Session

1. Visit any blocked website
2. You'll see the blocked page with a timer showing **25:00**
3. Click the **Start** button to begin your work session
4. The timer will count down from 25 minutes

- You get a 5-minute break
- The page shows this with a green indicator
- Blocked sites remain blocked during breaks (to help you truly rest!)

- When the 5-minute break ends, a new 25-minute work session automatically begins

## The Pomodoro Technique

This timer implements the classic **Pomodoro Technique**:
- **25 minutes** of focused work
- **5 minutes** of rest
- Repeat throughout your day

This technique is scientifically proven to improve focus and productivity while preventing burnout.
